# proof-of-work blockchain (flask)

a simple proof-of-work (pow) blockchain implementation built with python and flask.
this project demonstrates how decentralized nodes interact, validate transactions, and mine new blocks using pow consensus.

---

## features

* basic blockchain structure
* transaction handling
* proof-of-work mining
* peer-to-peer node communication
* consensus mechanism (longest chain rule)
* rest api using flask

---

## project structure

```
.
├── app.py

├── node.py

```

## notes

* this is a learning project, not production-ready
* no cryptographic wallets or signatures implemented
* networking is simplified
